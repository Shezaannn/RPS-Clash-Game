# RPS Clash 🎮

**RPS Clash** is a simple Android game based on the classic **Rock–Paper–Scissors** challenge.  
This project was created as a learning exercise to explore core Android development concepts using **Java and Android Studio**, including UI design, activity navigation, and game logic.

---

## 📱 Features

- Play **Rock, Paper, Scissors** against the CPU
- **Random CPU move generation**
- **Score tracking** for both player and CPU
- **Restart Game** button to reset scores
- Clean and minimal **modern UI**
- **Splash screen** when the app launches

---

## 🕹️ How to Play

1. Download the APK from Google Drive:  
   https://drive.google.com/file/d/1nk83cT0skTOK-cSjBj2vMO8ogCpCrUso/view?usp=drive_link

2. Install the APK on your Android device.

3. Open the **RPS Clash** app.

4. Choose **Rock**, **Paper**, or **Scissors**.

5. The CPU randomly selects its move.

6. The result of the round will be displayed on the screen.

7. Scores update automatically after each round.

8. Press **Restart Game** if you want to reset the scores.

---

## 🧠 Game Rules

- **Rock beats Scissors**
- **Scissors beats Paper**
- **Paper beats Rock**
- If both players choose the same option, the result is a **Draw**
